# Screenshots Directory

Place your FluxFlow screenshots here:

1. **dashboard.png** - Full dashboard view showing sidebar + kanban board
2. **sidebar.png** - Project sidebar with progress rings and drag handles
3. **kanban.png** - Kanban board with task cards in different columns
4. **task-details.png** - Task details slide-over panel with attachments
5. **search.png** - Global search modal with results

## How to Take Screenshots

### Windows
- Press `Win + Shift + S` to open Snipping Tool
- Select the area to capture
- Save as PNG

### Mac
- Press `Cmd + Shift + 4` to select area
- Press `Cmd + Shift + 3` for full screen

### Recommended Dimensions
- Dashboard: 1920x1080 or 2560x1440
- Individual features: 800x600 minimum
